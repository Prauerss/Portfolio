Snake NEAT

NeuroEvolution of Augmenting Topologies applied to Snake Game

C++ SFML NEAT Algorithm Neural Networks Genetic Algorithm

Project Overview

Implementation of the NEAT algorithm to train an AI to play Snake through neuroevolution

Concept & Vision

NEAT Snake is a project that implements the NeuroEvolution of Augmenting Topologies (NEAT) algorithm to train an artificial intelligence to play the classic Snake game. The system evolves neural networks through generations, selecting the best-performing individuals based on their ability to navigate the game, collect apples, and avoid collisions.

Key Features

NEAT Algorithm Implementation

Full implementation of the NEAT algorithm with speciation, crossover, and mutation operations to evolve neural network topologies.

Neural Network Architecture

Custom neural network implementation with configurable input, hidden, and output layers, using ReLU and Tanh activation functions.

Fitness Evaluation

Comprehensive fitness function that rewards apples collected, penalizes collisions, and encourages efficient movement toward targets.

Genome Serialization

Ability to save and load the best-performing genomes for continued training or demonstration purposes across different sessions.

Interactive Demonstration

Real-time visualization of the training process and the ability to watch the best AI play with configurable speed and generation selection.

Technical Challenges & Solutions

NEAT Algorithm Complexity

Algorithm Design
Challenge

Implementing the complete NEAT algorithm with speciation, historical markings, and complex crossover operations presented significant complexity in both design and implementation.

Solution

Developed a modular architecture with separate components for genome management, speciation, crossover, and mutation. Implemented historical markings using innovation numbers to track gene ancestry.

Performance Optimization

Computational Efficiency
Challenge

Evaluating hundreds of neural networks in real-time for each generation required significant computational resources, leading to slow training times.

Solution

Optimized neural network forward propagation, implemented efficient data structures for genome management, and added configurable population sizes and maximum step limits per evaluation.

Fitness Function Design

Algorithm Tuning
Challenge

Creating a fitness function that effectively rewarded desired behaviors (collecting apples, avoiding walls/self) without creating local optima or deceptive gradients.

Solution

Developed a multi-component fitness function with rewards for apples collected (with exponential bonuses for consecutive apples), penalties for collisions, and small rewards for moving toward the apple.

Genome Serialization

Data Persistence
Challenge

Saving and loading complex genome structures with variable topology sizes, connection genes, and node genes required a robust serialization system.

Solution

Implemented a binary serialization format that stores network dimensions, weights, biases, and metadata. Added versioning and validation to handle compatibility across different runs.
